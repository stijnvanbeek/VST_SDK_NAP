VST3 Project Generator

This small application (Win/macOS) allows you to generate easily a new VST 3 plug-in project by just entering in a GUI some parameters.

Check the folder "VST3_Project_Generator" of the SDK!

See https://developer.steinberg.help/display/VST/VST+3+Project+Generator for more information about how to use it.
